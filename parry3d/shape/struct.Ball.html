pub fn scaled(
    self,
    scale: &Vector<Real>,
    nsubdivs: u32
) -> Option<Either<Self, ConvexPolyhedron>>

Computes a scaled version of this ball.

If the scaling factor is non-uniform, then it can’t be represented as ball. Instead, a convex polygon approximation (with nsubdivs subdivisions) is returned. Returns None if that approximation had degenerate normals (for example if the scaling factor along one axis is zero).
